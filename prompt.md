# 思维树框架提示词：囚徒健身进度追踪网页

**Root（根）**：
设计一个基于《囚徒健身》六艺进度的网页应用，帮助用户追踪健身进度、记录数据并提供成就感分享功能

**Nodes（节点）**：
- **节点1：六艺进度系统设计**
  - 六艺分类（俯卧撑、深蹲、引体向上、举腿、桥、倒立撑）
  - 每艺的阶段划分（初、中、高级）
  - 阶段完成标准
  - 最终技定义与庆祝机制

- **节点2：数据记录系统**
  - 本地数据存储方案
  - 每日打卡功能
  - 进度追踪算法
  - 阶段完成时间计算

- **节点3：用户界面设计**
  - 首页展示（当前进度、已完成阶段）
  - 六艺详情页（各阶段动作列表、完成情况）
  - 打卡界面（日期选择、动作记录、完成度）
  - 统计页面（各阶段耗时、最终技完成情况）

- **节点4：分享卡片功能**
  - 卡片模板设计
  - 个人数据提取（当前阶段、耗时、完成最终技）
  - 分享渠道整合
  - 自定义选项

- **节点5：用户体验优化**
  - 初学者引导流程
  - 成就系统（徽章、进度条）
  - 提醒与激励机制
  - 社交互动功能（可选）

**Edges（边）**：
- 六艺进度系统 → 数据记录系统（阶段完成标准触发数据记录）
- 数据记录系统 → 用户界面设计（数据驱动界面展示）
- 用户界面设计 → 分享卡片功能（界面数据作为卡片内容来源）
- 六艺进度系统 → 用户体验优化（阶段进度触发成就系统）
- 所有节点 → 迭代优化循环（用户反馈驱动系统改进）

**Leaves（叶）**：
- 完整的网页应用设计方案
- 技术实现路线图
- 用户测试与反馈收集机制
- 上线后迭代优化计划
- 风险评估与应对措施
